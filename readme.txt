Projekt PAB - System dla miejskiej rowerowej firmy kurierskiej

Autorzy:
- Aleksandra Witkowska
- Justyna Wachowiak
- Tomasz Solarski

Działanie systemu:
- możliwość tworzenia klientów, kurierów oraz zamówień
- możliwość ustawienia zdjęcia profilowego dla kuriera
- możliwość zamawiania przesyłek przez klientów (formularz)
- możliwość śledzenia przesyłki
- możliwość wybierania przesyłek do zrealizowania przez kuriera
- możliwość zmiany stanu przesyłki (odebrana od klienta/dostarczona oraz czasu kiedy to się wydarzyło) przez kuriera
- możliwość zobaczenia edycji informacji o kurierze, kliencie, zamówieniu
- możliwość usuwania powyższych obiektów


Aby uruchomić program należy wybrać jako Startup Projects projekty: WebAPI, BlazorServer, BlazorClient.
